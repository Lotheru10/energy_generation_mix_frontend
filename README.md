# Energy Generation Mix - Frontend

Frontend for the Energy Generation Mix application for Great Britain

Live page: https://energy-mix.onrender.com

This application uses public API Carbon Intensity API to analyze daily UK's energy generation mix and find the most eco-friendly window to charge a car

## Features

### Daily Energy Generation Mix
- Displays daily average energy mix for today, tomorrow and the day after tomorrow
- Shows clean energy percentage per day
- Visualizes data with chars

## Best Charging Window
- Allows user to input a charging duration
- Dispalys the best charging time window based on used clean energy

## Tech stack
- React
- TypeScript
- Vite

## How to run locally
npm install
npm run dev
